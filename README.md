setup:

```
$pip3 install django

$django-admin startproject django-todo .
```

check setup:
```
$Python3 manage.py runserver
```
start building app:
```
$python3 manage.py startapp todo
```

make a local clone

```
$mkdir django-lessons
$git init
$git clone https://github.com/JorisPaarde/django-lessons.git
```
